# JavaScript Concepts

This folder contains structured and organized notes, examples, and code implementations for core and advanced JavaScript concepts.

The goal is to build a strong foundation in JavaScript by learning each concept deeply and applying it through code.

---

## 📁 Folder Structure

### 01 - Fundamentals

Covers the basics of JavaScript, including:

- Variables
- Data types
- Operators

---

### 02 - Arrays

Topics related to arrays, including:

- Destructuring arrays
- Array methods (map, filter, reduce)

---

### 03 - Objects

Focuses on JavaScript objects:

- Object destructuring
- Object methods
- Working with key-value data

---

### 04 - Advanced

Advanced JavaScript concepts such as:

- Spread operator
- Rest parameters
- Async/Await
- Closures (to be added)

---
